Cara Instalasi di HP Android

Karena aplikasi ini diunduh langsung (bukan dari Play Store), ikuti langkah mudah berikut:

1

Download File

Klik tombol Download di atas, dan simpan file ASMARA.apk di HP Anda.

2

Buka File

Buka notifikasi unduhan yang selesai, lalu ketuk file APK tersebut untuk membukanya.

3

Izinkan Akses

Jika muncul peringatan "Install unknown apps", klik Settings lalu aktifkan "Allow from this source".

4

Install & Mainkan!

Kembali ke layar instalasi, klik Install, tunggu beberapa detik, lalu klik Open.
